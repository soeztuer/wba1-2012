Was ist HTML?

  • HyperText Markup Language
  • Markup Language -> Auszeichnungssprache
  • Aktuelle Version: HTML5

Geschichte von HTML

  • 1989 - Erfindung in CERN von Tim Beners-Lee
  • 1992 - Offizielles Release von HTML
  • 1993 - Release der ersten Erweiterung
  • 1997 - Release von HTML3.
  • 1998 - HTML4.0 mit Verweisen auf CSS
  • 2014 - Offentliches Release von HTML5

Head

  • <title>-Tag -> Im Reiter angezeiger Name
  • Informationen für Suchmaschinen
  • Beschreibung der Seite
  • Sprache der Seite
  • Javascript
  • Verweis auf CSS

Doctype

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"
       "http://www.w3.org/TR/html4/loose.dtd">
Gibt HTML-Version und Variante an (loose/strict). 

<!DOCTYPE html>
Der HTML5 Doctype ist wesentlich simpler.

<div> und <span>

<div> wird zum Sektionieren und Einteilen von Block-Elementen in einem HTML Dokumenten verwendet.
<span> wird standartmäßig für Inline-Elemente verwendet.

Beide Tags werden dazu verwendet um die Website nach eigenen Wünschen mit CSS zu gestalten.

Die Einteilung einer Seite

<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN"
       "http://www.w3.org/TR/html4/loose.dtd">
<html>
    <head>
        <title>Beschreibung der Seite</title>
    </head>
    <body>
	 <div id="header">
	
	 </div>
	
	 <div id="navigation">
	 </div>
	
	 <div id="main">
	 </div>
    </body>
</html>

Schriftstile

  • Fett: <b> oder <strong>
  • Kursiv: <i> oder <em>
  • Unterstrichen: <u>
